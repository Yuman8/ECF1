HOROSCOPE

Bienvenue sur le site HOROSCOPE

Dans le cadre de l’ECF1 consacré au développement front-end, il nous a été confié la mission de réaliser l’interface d’une application d’horoscope. Ce projet a pour objectif de mettre en valeur les compétences acquises jusqu’à présent en HTML sémantique, CSS, JavaScript, animations, dynamisme et interactions avec une API.

Sur cette page, un appel à une API permet de récupérer les données des 12 signes astrologiques. Lorsque vous cliquez sur le bouton « Afficher les 12 cartes », celles-ci s’affichent dynamiquement. Passez sur chaque image pour découvrir le message associé à chaque signe.
